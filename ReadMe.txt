IDBDocumentor

This utility documents a SQL Server 2000 database, in to Microsoft compiled html help format. It Documents the following database objects.

1.	Tables (Table Description, Columns & Column Descriptions, References)
2.	Triggers (Properties, source, description)
3.	Stored procedures (Properties, source, description)
4.	User Defined Functions (Properties, source, description)
5.	Views (Columns & Column Descriptions, References)


This Utility requires following software's to be installed
1.	Microsoft SQL Server 2000(Server/Client) SP1 or greater to be installed on the client machine.
2.	HTML Help Workshop
3.	Visual Basic Runtime
4.	Microsoft internet explorer 5.0 or greater


HTML Help Workshop can be downloaded from following sites.

1.	http://download.microsoft.com/download/OfficeXPProf/Install/4.71.1015.0/W98NT42KMe/EN-US/HTMLHELP.EXE
2.	http://msdn.microsoft.com/library/en-us/htmlhelp/html/hwMicrosoftHTMLHelpDownloads.asp?frame=true
